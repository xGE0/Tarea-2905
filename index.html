¿Qué es JSON?

JSON (JavaScript Object Notation) es un formato de texto ligero para el intercambio de datos. Es fácil para los humanos leer y escribir, y fácil para las máquinas parsear y generar. Se utiliza principalmente para transmitir datos entre un servidor y una aplicación web, como una alternativa a XML.

¿Qué es una API?

API (Interfaz de Programación de Aplicaciones) es un conjunto de definiciones y protocolos que permiten la comunicación entre diferentes software. Las APIs se utilizan para que diferentes sistemas, aplicaciones y servicios puedan interactuar y compartir datos entre sí.

¿Qué es un Web Service?

Un Web Service es un método de comunicación entre dos dispositivos a través de una red. Los web services permiten a diferentes aplicaciones intercambiar datos y servicios entre sí, sin importar en qué plataforma estén basadas. Utilizan estándares como XML, JSON, SOAP y REST para la comunicación.
